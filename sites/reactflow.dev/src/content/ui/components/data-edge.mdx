# 数据边

一种显示源节点`data`对象中某个字段的边。

<UiComponentViewer id="data-edge" />

## 额外类型安全

创建此类新边时，可以使用TypeScript的`satisfies`谓词
结合应用中节点的具体类型，确保边的数据`key`
属性是节点数据的有效键名。

```ts
type CounterNode = Node<{ count: number }>;

const initialEdges = [
  {
    id: 'edge-1',
    source: 'node-1',
    target: 'node-2',
    type: 'dataEdge',
    data: {
      key: 'count',
    } satisfies DataEdge<CounterNode>,
  },
];
```

如果尝试使用节点数据中不存在的键名，TypeScript会
显示如下错误信息：

> ts: 类型'"value"'不能赋值给类型'"count"'。
