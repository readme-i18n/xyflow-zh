# 动画SVG边

一种沿边缘路径动画化自定义SVG元素的边。该组件基于[动画化SVG元素示例](/examples/edges/animating-edges)。

<UiComponentViewer id="animated-svg-edge" />

## 自定义形状

建议您向模块添加自己的SVG形状。每个形状应是一个接收`animateMotionProps`属性并返回SVG的React组件。

您可以在单独文件中定义这些形状，也可以与边组件定义在同一文件中。要使用它们，需要像这样将它们添加到`shapes`记录中：

```tsx
const shapes = {
  box: ({ animateMotionProps }) => (
    <rect width="5" height="5" fill="#ff0073">
      <animateMotion {...animateMotionProps} />
    </rect>
  ),
} satisfies Record<string, AnimatedSvg>;
```

`shapes`记录的键是边数据中`shape`字段的有效值：

```ts
const initialEdges = [
  {
    // ...
    type: "animatedSvgEdge",
    data: {
      duration: 2,
      shape: "box",
    },
  } satisfies AnimatedSvgEdge,
];
```

<Callout type="info">
  如果要渲染常规HTML元素，请确保将它们包裹在SVG的`<foreignObject />`元素中。务必给`<foreignObject />`添加`id`属性，并在渲染`<animateMotion />`元素时将其用作`href`属性。
</Callout>
