# 节点状态指示器

该组件是一个节点包装器，可通过多种状态来显示节点状态。支持的状态包括：`"success"`、`"loading"`、`"error"` 和 `"initial"`。

`NodeStatusIndicator` 组件还支持不同的加载变体：通过 `loadingVariant` 属性可设置为 `"border"` 或 `"overlay"`。

- `"border"` 变体为默认选项，在加载状态下会显示环绕节点的旋转边框
- `"overlay"` 变体在加载状态下会显示全屏覆盖层，并在节点上展示动画旋转图标
