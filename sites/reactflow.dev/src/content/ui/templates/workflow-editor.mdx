# 工作流编辑器

工作流编辑器模板是一个基于 Next.js 的应用程序，旨在帮助您快速创建、管理和可视化工作流程。该项目使用 [React Flow UI](/ui) 构建，并采用 [Tailwind CSS](https://tailwindcss.com/) 和 [shadcn/ui](https://ui.shadcn.com/) 进行样式设计，为构建和扩展工作流编辑器提供了高度可定制的基础。

## 技术栈

- **React Flow UI**: 项目使用 [React Flow UI](/ui) 构建节点组件，这些组件旨在帮助您快速上手项目开发。

- **shadcn CLI**: 项目使用 [shadcn CLI](https://ui.shadcn.com/docs/cli) 管理 UI 组件。该工具基于 [Tailwind CSS](https://tailwindcss.com/) 和 [shadcn/ui](https://ui.shadcn.com/) 组件构建，可轻松添加和自定义 UI 元素。

- **使用 Zustand 进行状态管理**: 应用采用 Zustand 进行状态管理，为节点、连线及其他工作流相关数据提供了简单高效的状态管理方案。

## 功能特性

- **自动布局**：使用 [ELKjs](https://github.com/kieler/elkjs) 布局引擎自动排列节点与连线。
- **拖拽侧边栏**：通过拖拽机制添加和排列节点。
- **可定制组件**：利用 React Flow UI 和 shadcn 库创建高度可定制的节点与连线。
- **暗黑模式**：通过 Zustand 状态管理实现明暗主题切换。
- **运行器功能**：通过工作流运行器顺序执行并监控节点。
