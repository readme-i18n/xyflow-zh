# React Flow UI

基于 [shadcn/ui](https://ui.shadcn.com/) 组件和 [Tailwind CSS](https://tailwindcss.com/) 构建的开箱即用 React Flow 组件。适用于新项目、MVP 或需要快速启动的场景。

## 前置条件

您的项目需要**已配置 shadcn 和 tailwind**。若尚未安装，可按照 [shadcn 安装指南](https://ui.shadcn.com/docs/installation) 的步骤操作。若项目中已包含 shadcn 和 tailwind，可通过以下命令初始化 shadcn-ui：

```bash copy npm2yarn
npx shadcn@latest init
```

如需了解本项目的设计初衷，可查阅详细博文 [此处](https://xyflow.com/blog/react-flow-components)。我们还发布了 [React Flow UI 入门指南](/learn/tutorials/getting-started-with-react-flow-components) 提供更深入的教程。

## 使用指南

选择所需组件后运行命令即可添加到项目。

```bash copy npm2yarn
npx shadcn@latest add https://ui.reactflow.dev/component-name
```

- 该命令会将组件代码复制到您的组件目录（可通过 `components.json` 中的别名修改目标目录）
- 自动安装所有必要依赖项
- 智能复用已添加或修改过的组件，或询问是否覆盖
- 兼容现有 tailwind 配置
- 组件**非黑盒设计**，可**自由修改扩展**以满足需求

更多信息请访问 [shadcn 文档](https://ui.shadcn.com/docs)。
