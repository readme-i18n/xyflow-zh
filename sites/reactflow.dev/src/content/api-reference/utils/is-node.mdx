# isNode()

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/utils/graph.ts/#L49)

测试一个对象是否可用作 [`Node`](/api-reference/types/node)。在 TypeScript 中，这是一个类型守卫，当返回值为 `true` 时，会将传入参数的类型收窄为 [`Node`](/api-reference/types/node)。

```js
import { isNode } from '@xyflow/react';

const node = {
  id: 'node-a',
  data: {
    label: 'node',
  },
  position: {
    x: 0,
    y: 0,
  },
};

if (isNode(node)) {
  // ..
}
```
