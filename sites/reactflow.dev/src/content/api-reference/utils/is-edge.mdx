# isEdge()

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/utils/graph.ts/#L39-L40)

测试一个对象是否可用作 [`Edge`](/api-reference/types/edge)。在 TypeScript 中，这是一个类型守卫，当返回 `true` 时会将传入值的类型收窄为 [`Edge`](/api-reference/types/edge)。

```js
import { isEdge } from '@xyflow/react';

const edge = {
  id: 'edge-a',
  source: 'a',
  target: 'b',
};

if (isEdge(edge)) {
  // ...
}
```
