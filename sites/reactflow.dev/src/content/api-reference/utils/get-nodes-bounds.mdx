# getNodesBounds()

[GitHub 源代码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/utils/graph.ts/#L133)

返回包含数组中所有给定节点的边界框。当与 [`getViewportForBounds`](/api-reference/utils/get-viewport-for-bounds) 结合使用时，可用于计算正确的变换以使给定节点适应视口。

> [!NOTE]
>
> 该函数之前名为 `getRectOfNodes`

```js
import { getNodesBounds } from '@xyflow/react';

const nodes = [
  {
    id: 'a',
    position: { x: 0, y: 0 },
    data: { label: 'a' },
    width: 50,
    height: 25,
  },
  {
    id: 'b',
    position: { x: 100, y: 100 },
    data: { label: 'b' },
    width: 50,
    height: 25,
  },
];

const bounds = getNodesBounds(nodes);
```
