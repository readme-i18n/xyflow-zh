# applyEdgeChanges()

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/utils/changes.ts/#L167)

[`<ReactFlow />`](/api-reference/react-flow) 组件上的各种事件可能会产生 [`EdgeChange`](/api-reference/types/edge-change)，用于描述如何以某种方式更新流程中的边。如果不需要自定义行为，可以使用此工具函数接收这些变更数组并将其应用到边的更新中。

```js
import { useState, useCallback } from 'react';
import { ReactFlow, applyEdgeChanges } from '@xyflow/react';

export default function Flow() {
  const [nodes, setNodes] = useState([]);
  const [edges, setEdges] = useState([]);
  const onEdgesChange = useCallback(
    (changes) => {
      setEdges((oldEdges) => applyEdgeChanges(changes, oldEdges));
    },
    [setEdges],
  );

  return (
    <ReactFlow nodes={nodes} edges={edges} onEdgesChange={onEdgesChange} />
  );
}
```

## 注意事项
