# useInternalNode()

[GitHub源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/hooks/useInternalNode.ts)

该钩子返回特定节点的内部表示。使用此钩子的组件
会在**任何节点发生变化时重新渲染**，包括节点被选中
或移动时。

```jsx
import { useInternalNode } from '@xyflow/react';

export default function () {
  const internalNode = useInternalNode('node-1');
  const absolutePosition = internalNode.internals.positionAbsolute;

  return (
    <div>
      The absolute position of the node is at:
      <p>x: {absolutePosition.x}</p>
      <p>y: {absolutePosition.y}</p>
    </div>
  );
}
```

## TypeScript

该钩子接受自定义节点类型的泛型参数。更多信息请参阅
[TypeScript指南中的相关章节](/learn/advanced-use/typescript#nodetype-edgetype-unions)。

```tsx
const internalNode = useInternalNode<CustomNodeType>();
```
