# useViewport()

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/hooks/useViewport.ts)

`useViewport` 钩子提供了一种便捷方式，用于在组件中读取当前 [`Viewport`](/api-reference/types/viewport) 状态。使用此钩子的组件**会在视口变化时**重新渲染。

```jsx
import { useViewport } from '@xyflow/react';

export default function ViewportDisplay() {
  const { x, y, zoom } = useViewport();

  return (
    <div>
      <p>
        The viewport is currently at ({x}, {y}) and zoomed to {zoom}.
      </p>
    </div>
  );
}
```

## 签名

<APIDocs functionName="useViewport" />

## 注意事项

- 该钩子只能在 [`<ReactFlowProvider />`](/api-reference/react-flow-provider) 或 [`<ReactFlow />`](/api-reference/react-flow) 组件的子组件中使用。
