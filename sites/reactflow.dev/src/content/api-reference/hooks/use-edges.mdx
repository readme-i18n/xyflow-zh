# useEdges()

[GitHub源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/hooks/useEdges.ts)

该钩子返回当前边的数组。使用此钩子的组件会在**任何边发生变化**时重新渲染。

```jsx
import { useEdges } from '@xyflow/react';

export default function () {
  const edges = useEdges();

  return <div>There are currently {edges.length} edges!</div>;
}
```

## 签名

<APIDocs functionName="useEdges" />

## TypeScript

该钩子接受自定义边类型的泛型参数。更多信息请参阅[TypeScript指南中的相关章节](/learn/advanced-use/typescript#nodetype-edgetype-unions)。

```tsx
const nodes = useEdges<CustomEdgeType>();
```

## 注意事项

- 不必要地依赖`useEdges`可能导致性能问题。任何边发生变化时，该钩子都会触发组件重新渲染。实际上我们通常更关心特定变化（如边的_数量_变化）：尽可能尝试改用[`useStore`](/api-reference/hooks/use-store)。
