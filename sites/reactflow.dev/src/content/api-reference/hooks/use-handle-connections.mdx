# useHandleConnections()

> [!WARNING]
>
> `useHandleConnections` 已被弃用，建议使用功能更强大的 [useNodeConnections](/api-reference/hooks/use-node-connections)。

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/hooks/useHandleConnections.ts)

该钩子返回特定连接点或连接点类型的连接数组。

```jsx
import { useHandleConnections } from '@xyflow/react';

export default function () {
  const connections = useHandleConnections({ type: 'target', id: 'my-handle' });

  return (
    <div>There are currently {connections.length} incoming connections!</div>
  );
}
```
