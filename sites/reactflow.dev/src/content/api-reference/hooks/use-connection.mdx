# useConnection()

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/hooks/useConnection.ts)

`useConnection` 钩子会在存在活跃连接交互时返回当前连接状态。若无活跃连接交互，则所有属性均返回 `null`。该钩子的典型应用场景是根据特定条件（例如连接是否有效）为连接手柄着色。

```tsx
import { useConnection } from '@xyflow/react';

export default function App() {
  const connection = useConnection();

  return (
    <div>
      {connection ? `Someone is trying to make a connection from ${connection.fromNode} to this one.` : 'There are currently no incoming connections!'}
    </div>
  );
}
```
