# \<ReactFlow />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/container/ReactFlow/index.tsx/#L47)

`<ReactFlow />` 组件是 React Flow 应用的核心。它负责渲染节点和边、处理用户交互，当作为[非受控流程图](/learn/advanced-use/uncontrolled-flow)使用时还能管理自身状态。

```tsx
import { ReactFlow } from '@xyflow/react'

export default function Flow() {
  return <ReactFlow
    nodes={...}
    edges={...}
    onNodesChange={...}
    ...
  />
}
```

该组件接收大量不同的属性参数，其中大多数为可选。我们已按功能分类进行文档整理，方便您快速查阅。

## 键盘属性

React Flow 允许您传入几种不同的键盘快捷键，作为与流程图交互的另一种方式。我们已尝试设置合理的默认值，例如使用退格键删除选定的节点或边，但您可以通过这些属性来自定义设置。

若要禁用任何快捷键，只需将 `null` 传递给要禁用的对应属性即可。

<ReactFlowAPIProps group="keyboard" />
