## 致JavaScript用户的说明

React Flow虽采用TypeScript编写，但我们理解并非所有开发者都使用它。我们鼓励开发者选用最适合的技术方案，因此在文档中会同时呈现TypeScript和JavaScript的示例代码。

然而，在我们的 API 参考文档中，我们使用 TypeScript 语法来记录属性和函数的类型。以下是快速理解这些语法的指南：

• `?` 表示该字段或参数是可选的。

• 类型定义中的 `<T>` 表示泛型类型参数。就像函数的参数，但用于类型！定义 `type Array<T> = ...` 表示一个名为 `Array` 的类型，它接受泛型类型参数 `T`。

• 当引用类型时，`<T>` 类似于"填充"泛型类型参数。就像调用函数，但针对类型！类型 `Array<number>` 表示用 `number` 类型填充了泛型参数 `T` 的 `Array` 类型。

• `T | U` 表示该类型可以是 `T` 或 `U`：这通常被称为 _联合类型_。

• `T & U` 表示该类型同时是 `T` 和 `U`：这通常被称为 _交叉类型_。
