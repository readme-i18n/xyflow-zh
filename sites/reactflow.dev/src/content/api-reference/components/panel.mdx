# \<Panel />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/components/Panel/index.tsx)

`<Panel />` 组件可帮助您在视口上方定位内容，被 [`<MiniMap />`](/api-reference/components/minimap) 和 [`<Controls />`](/api-reference/components/controls) 组件内部使用。

```jsx
import { ReactFlow, Panel } from '@xyflow/react';

export default function Flow() {
  return (
    <ReactFlow nodes={[...]} fitView>
      <Panel position="top-left">top-left</Panel>
      <Panel position="top-center">top-center</Panel>
      <Panel position="top-right">top-right</Panel>
      <Panel position="bottom-left">bottom-left</Panel>
      <Panel position="bottom-center">bottom-center</Panel>
      <Panel position="bottom-right">bottom-right</Panel>
    </ReactFlow>
  );
}
```

## Props 属性

对于 TypeScript 用户，`<Panel />` 组件的 props 类型导出为 `PanelProps`。此外，`<Panel />` 组件接受所有 HTML `<div />` 元素的属性。
