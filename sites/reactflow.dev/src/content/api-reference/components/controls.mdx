# \<Controls />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/additional-components/Controls/Controls.tsx)

`<Controls />` 组件会渲染一个包含便捷按钮的小面板，功能包括放大、缩小、适应视图以及锁定视口。

```tsx
import { ReactFlow, Controls } from '@xyflow/react'

export default function Flow() {
  return (
    <ReactFlow nodes={[...]} edges={[...]}>
      <Controls />
    </ReactFlow>
  )
}
```

## Props 属性

对于 TypeScript 用户，`<Controls />` 组件的属性类型以 `ControlProps` 形式导出。

<APIDocs componentName="Controls" />

## 注意事项

- 如需扩展或自定义控制按钮，可使用 [`<ControlButton />`](/api-reference/components/control-button) 组件
