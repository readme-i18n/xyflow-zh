# \<Background />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/additional-components/Background/Background.tsx)

`<Background />` 组件可便捷渲染基于节点的用户界面中常见的各类背景，提供三种变体：`lines`（线条）、`dots`（圆点）和 `cross`（十字网格）。

```jsx
import { useState } from 'react';
import { ReactFlow, Background, BackgroundVariant } from '@xyflow/react';

export default function Flow() {
  return (
    <ReactFlow defaultNodes={[...]} defaultEdges={[...]}>
      <Background color="#ccc" variant={BackgroundVariant.Dots} />
    </ReactFlow>
  );
}
```

## 使用示例

### 组合多重背景

可通过叠加多个 `<Background />` 组件来创建更丰富的视觉效果。以下示例展示了如何渲染每10条线高亮显示的方形网格。

```tsx
import { ReactFlow, Background, BackgroundVariant } from '@xyflow/react';

import '@xyflow/react/dist/style.css';

export default function Flow() {
  return (
    <ReactFlow defaultNodes={[...]} defaultEdges={[...]}>
      <Background
        id="1"
        gap={10}
        color="#f1f1f1"
        variant={BackgroundVariant.Lines}
      />

      <Background
        id="2"
        gap={100}
        color="#ccc"
        variant={BackgroundVariant.Lines}
      />
    </ReactFlow>
  );
}
```

## 注意事项

- 组合多个 `<Background />` 组件时，必须为每个组件设置唯一的 `id` 属性！
