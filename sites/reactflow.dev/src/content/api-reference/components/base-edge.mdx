# \<BaseEdge />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/components/Edges/BaseEdge.tsx)

`<BaseEdge />` 组件在内部被用于所有边的渲染。它可以在自定义边中使用，并自动处理不可见的辅助边和边标签。

```jsx
import { BaseEdge } from '@xyflow/react';

export function CustomEdge({ sourceX, sourceY, targetX, targetY, ...props }) {
  const [edgePath] = getStraightPath({
    sourceX,
    sourceY,
    targetX,
    targetY,
  });

  const { label, labelStyle, markerStart, markerEnd, interactionWidth } = props;

  return (
    <BaseEdge
      path={edgePath}
      label={label}
      labelStyle={labelStyle}
      markerEnd={markerEnd}
      markerStart={markerStart}
      interactionWidth={interactionWidth}
    />
  );
}
```

## Props 属性

<APIDocs
  componentName="BaseEdge"
  groupKeys="Omit<SVGAttributes<SVGPathElement>, 'd' | 'path' | 'markerStart' | 'markerEnd'>"
/>

## 注意事项

- 若需在 [`<BaseEdge />`](/api-reference/components/base-edge) 组件中使用边标记，可将自定义边接收到的 `markerStart` 或 `markerEnd` 属性透传给该组件。通过查看 [`EdgeProps`](/api-reference/types/edge-props) 类型可获取传递给自定义边的所有属性。
