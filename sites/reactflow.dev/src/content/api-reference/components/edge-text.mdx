# \<EdgeText />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/components/Edges/EdgeText.tsx)

您可以使用 `<EdgeText />` 组件作为辅助组件，在自定义边线上显示文本。

```jsx
import { EdgeText } from '@xyflow/react';

export function CustomEdgeLabel({ label }) {
  return (
    <EdgeText
      x={100}
      y={100}
      label={label}
      labelStyle={{ fill: 'white' }}
      labelShowBg
      labelBgStyle={{ fill: 'red' }}
      labelBgPadding={[2, 4]}
      labelBgBorderRadius={2}
    />
  );
}
```

## Props 属性

对于 TypeScript 用户，`<EdgeText />` 组件的属性类型已导出为 `EdgeTextProps`。

<APIDocs componentName="EdgeText" groupKeys="Omit<SVGAttributes<SVGElement>, 'x' | 'y'>" />

此外，您还可以传递任何标准的 React HTML 属性，例如 `onClick`、`className` 等。
