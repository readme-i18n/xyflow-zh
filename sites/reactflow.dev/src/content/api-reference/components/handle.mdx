# \<Handle />

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/components/Handle/index.tsx)

`<Handle />` 组件用于在[自定义节点](/learn/customization/custom-nodes)中定义连接点。

```jsx
import { Handle, Position } from '@xyflow/react';

export const CustomNode = ({ data }) => {
  return (
    <>
      <div style={{ padding: '10px 20px' }}>
        {data.label}
      </div>

      <Handle type="target" position={Position.Left} />
      <Handle type="source" position={Position.Right} />
    </>
  );
};
```

## Props 属性

对于 TypeScript 用户，`<Handle />` 组件的属性类型以 `HandleProps` 形式导出。
