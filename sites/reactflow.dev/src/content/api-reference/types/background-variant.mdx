# BackgroundVariant

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/additional-components/Background/types.ts)

为了方便使用，这三种变体以枚举形式导出。您既可以导入枚举并使用类似 `BackgroundVariant.Lines` 的方式调用，也可以直接使用原始字符串值。

```ts
export enum BackgroundVariant {
  Lines = 'lines',
  Dots = 'dots',
  Cross = 'cross',
}
```
