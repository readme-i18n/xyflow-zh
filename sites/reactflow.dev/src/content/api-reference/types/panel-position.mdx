# PanelPosition

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts/#L111-L112)

该类型主要用于帮助在流式视口上方定位元素。例如，[`<MiniMap />`](/api-reference/components/minimap) 和 [`<Controls />`](/api-reference/components/controls) 组件都接受此类型的 `position` 属性。

```ts
export type PanelPosition =
  | 'top-left'
  | 'top-center'
  | 'top-right'
  | 'bottom-left'
  | 'bottom-center'
  | 'bottom-right';
```
