# SelectionMode

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts#L223)

`SelectionMode` 枚举提供了两种节点选择行为选项：

- `Full`: 仅当选择矩形完全包含节点时才选中该节点
- `Partial`: 当选择矩形与节点部分重叠时即选中该节点

```ts
enum SelectionMode {
  Partial = 'partial',
  Full = 'full',
}
```
