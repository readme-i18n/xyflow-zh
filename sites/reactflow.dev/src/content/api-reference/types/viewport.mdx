# 视口

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts/#L149-L153)

在内部，React Flow 维护着一个独立于页面其他部分的坐标系。`Viewport` 类型会告诉你当前流程在该坐标系中的显示位置以及缩放级别。

## 注意事项
