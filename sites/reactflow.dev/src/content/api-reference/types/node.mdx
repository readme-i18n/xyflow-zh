# 节点

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/nodes.ts/#L10)

`Node` 类型定义了 React Flow 需要了解的节点全部信息。这些属性大多可由 React Flow 或用户双向操作，但 `width` 和 `height` 等部分属性应视为只读。

## 字段说明

<APIDocs typeName="Node" packageName="react" />

## 默认节点类型

通过设置 `type` 属性为以下值之一，可以创建 React Flow 的任意默认节点：

- `"default"`
- `"input"`
- `"output"`
- `"group"`

若不设置 `type` 属性，React Flow 将回退为同时具备输入输出端口的 `"default"` 节点。

即使设置了 [`nodeTypes`](/api-reference/react-flow#node-types) 属性，这些默认节点仍然可用——除非直接覆盖了这些键值。

## 注意事项

- 不应直接设置节点的 `width` 或 `height`。这些值由 React Flow 内部计算，用于节点在视口中的渲染。如需控制节点尺寸，应通过 `style` 或 `className` 属性应用 CSS 样式来实现。
