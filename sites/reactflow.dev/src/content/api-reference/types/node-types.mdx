# NodeTypes

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/react/src/types/nodes.ts)

`NodeTypes` 类型用于定义自定义节点类型。对象中的每个键代表一个节点类型，对应的值是该类型节点应渲染的组件。

```ts
type NodeTypes = {
  [key: string]: React.ComponentType<NodeProps>;
};
```
