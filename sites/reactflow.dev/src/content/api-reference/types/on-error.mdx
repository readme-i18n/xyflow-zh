# OnError

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts#L233)

`OnError` 类型定义了在发生错误时调用的回调函数。该回调函数接收错误 ID 和错误消息作为参数。

```ts
type OnError = (id: string, error: string) => void;
```
