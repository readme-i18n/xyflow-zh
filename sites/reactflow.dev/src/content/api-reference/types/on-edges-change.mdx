# OnEdgesChange

该类型用于为 [`onEdgesChange`](/api-reference/react-flow#on-edges-change) 函数提供类型定义。

```tsx
export type OnEdgesChange<EdgeType extends Edge = Edge> = (
  changes: EdgeChange<EdgeType>[],
) => void;
```

## 使用指南

此类型接受自定义边类型的泛型参数。更多信息请参阅
[Typescript 指南中的相关章节](/learn/advanced-use/typescript#nodetype-edgetype-unions)。

```tsx
const onEdgesChange: OnEdgesChange = useCallback(
  (changes) => setEdges((eds) => applyEdgeChanges(changes, eds)),
  [setEdges],
);
```
