# DeleteElements

DeleteElements 删除提供的节点和边，并处理删除任何连接的边以及子节点。异步返回成功删除的边和节点。

```ts
export type DeleteElements = (payload: {
  nodes?: (Partial<Node> & { id: Node['id'] })[];
  edges?: (Partial<Edge> & { id: Edge['id'] })[];
}) => Promise<{
  deletedNodes: Node[];
  deletedEdges: Edge[];
}>;
```
