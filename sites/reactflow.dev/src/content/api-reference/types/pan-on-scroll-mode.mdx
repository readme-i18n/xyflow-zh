# PanOnScrollMode

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts#L166)

`PanOnScrollMode` 枚举控制用户滚动时视口的平移行为。选择 `Free` 表示无限制平移，`Vertical` 表示仅上下平移，`Horizontal` 表示仅左右平移。

```ts
enum PanOnScrollMode {
  Free = 'free',
  Vertical = 'vertical',
  Horizontal = 'horizontal',
}
```
