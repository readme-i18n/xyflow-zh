# OnNodeDrag

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/react/src/types/nodes.ts#L34)

`OnNodeDrag` 类型定义了当节点被拖动时调用的回调函数。该回调会接收事件对象和被拖动的节点。

```ts
type OnNodeDrag = (event: React.MouseEvent, node: Node) => void;
```
