# NodeHandle

[Source on GitHub](https://github.com/xyflow/xyflow/blob/13897512d3c57e72c2e27b14ffa129412289d948/packages/system/src/types/nodes.ts#L139)

`NodeHandle` 类型用于在服务端渲染时定义节点的连接点。由于服务端无法测量 DOM 节点，因此需要明确定义连接点的位置尺寸。
