# ConnectionLineComponent

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/types/edges.ts#L265)

`ConnectionLineComponent` 类型允许你提供一个自定义 React 组件，用于在用户创建新边时渲染连接线。该组件接收 `ConnectionLineComponentProps` 作为其 props。

```ts
type ConnectionLineComponent = React.ComponentType<ConnectionLineComponentProps>;
```
