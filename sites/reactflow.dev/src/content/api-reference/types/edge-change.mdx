# EdgeChange

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/changes.ts/#L68-L72)

[`onEdgesChange`](/api-reference/react-flow#on-edges-change) 回调函数接收一个 `EdgeChange` 对象数组，用于更新流程状态。`EdgeChange` 类型是四种不同对象类型的联合，表示流程中边(edge)可能发生的各种变化方式。

```ts
export type EdgeChange =
  | EdgeAddChange
  | EdgeRemoveChange
  | EdgeReplaceChange
  | EdgeSelectionChange;
```
