# OnNodesChange

该类型用于为 [`onNodesChange`](/api-reference/react-flow#on-nodes-change) 函数提供类型定义。

```tsx
export type OnNodesChange<NodeType extends Node = Node> = (
  changes: NodeChange<NodeType>[],
) => void;
```

## 使用指南

此类型接受一个泛型参数，用于指定自定义节点类型。更多信息请参阅
[TypeScript指南中的相关章节](/learn/advanced-use/typescript#nodetype-edgetype-unions)。

```tsx
const onNodesChange: OnNodesChange = useCallback(
  (changes) => setNodes((nds) => applyNodeChanges(changes, nds)),
  [setNodes],
);
```
