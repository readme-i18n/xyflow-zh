# ConnectionMode

[GitHub 源代码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts#L68)

`ConnectionMode` 枚举为 React Flow 中的连接行为提供了两种选项：

- `Strict`(严格模式)：连接只能从源句柄开始并在目标句柄结束
- `Loose`(宽松模式)：可以在任意类型的句柄之间建立连接

```ts
enum ConnectionMode {
  Strict = 'strict',
  Loose = 'loose',
}
```
