# NodeProps\<T>

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/nodes.ts/#L89)

当您实现[自定义节点](/learn/customization/custom-nodes)时，它会被包裹在一个提供基础功能（如选择和拖拽）的组件中。

## 使用指南

```tsx
import { useState } from 'react';
import { NodeProps, Node } from '@xyflow/react';

export type CounterNode = Node<
  {
    initialCount?: number;
  },
  'counter'
>;

export default function CounterNode(props: NodeProps<CounterNode>) {
  const [count, setCount] = useState(props.data?.initialCount ?? 0);

  return (
    <div>
      <p>Count: {count}</p>
      <button className="nodrag" onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}
```

请记得通过将自定义节点添加到 `<ReactFlow />` 组件的 [`nodeTypes`](/api-reference/react-flow#nodetypes) 属性中来注册它。

```tsx
import { ReactFlow } from '@xyflow/react';
import CounterNode from './CounterNode';

const nodeTypes = {
  counterNode: CounterNode,
};

export default function App() {
  return <ReactFlow nodeTypes={nodeTypes} ... />
}
```

您可以在我们的[自定义节点指南](/learn/customization/custom-nodes)中了解更多信息。
