# 边(Edge)

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/types/edges.ts/#L34-L353)

[`Connection`](/api-reference/types/connection) 是描述两个节点间连边的最小化定义，而 `Edge` 则是包含 React Flow 渲染所需全部信息的完整定义。

```ts
export type Edge<T> = DefaultEdge<T> | SmoothStepEdge<T> | BezierEdge<T>;
```

## 默认边类型

通过将 `type` 属性设置为以下值之一，可以创建 React Flow 的默认边类型：

- `"default"`
- `"straight"`
- `"step"`
- `"smoothstep"`
- `"simplebezier"`

如果不设置 `type` 属性，React Flow 将回退到默认的贝塞尔曲线边类型 `"default"`。

即使你将 [`edgeTypes`](/api-reference/react-flow#edge-types) 属性设置为其他值，这些默认边类型仍然可用，除非你直接覆盖了其中的任何键。
