# NodeChange

[GitHub 源码](https://github.com/xyflow/xyflow/blob/487b13c9ad8903789f56c6fcfd8222f9cb74b812/packages/system/src/types/changes.ts/#L47)

[`onNodesChange`](/api-reference/react-flow#on-nodes-change) 回调接收一个 `NodeChange` 对象数组，用于更新流程状态。`NodeChange` 类型是六种不同对象类型的联合，表示节点在流程中可能发生的各种变化。

```ts
export type NodeChange =
  | NodeDimensionChange
  | NodePositionChange
  | NodeSelectionChange
  | NodeRemoveChange
  | NodeAddChange
  | NodeReplaceChange;
```
