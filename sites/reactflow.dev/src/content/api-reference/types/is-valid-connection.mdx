# IsValidConnection

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/types/general.ts#L212)

`IsValidConnection` 类型表示一个验证节点间连接是否被允许的函数。它接收一个 [`Connection`](/api-reference/types/connection) 参数并返回布尔值，表示该连接是否有效因而应该被创建。

```ts
type IsValidConnection = (edge: Edge | Connection) => boolean;
```
