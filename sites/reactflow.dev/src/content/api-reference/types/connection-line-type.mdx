# ConnectionLineType

[GitHub 源码](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/edges.ts/#L62)

如果在 [`<ReactFlow />`](/api-reference/react-flow#connection-connectionLineType) 组件上设置 `connectionLineType` 属性，它将决定创建新边时渲染的连接线样式。

```ts
export enum ConnectionLineType {
  Bezier = 'default',
  Straight = 'straight',
  Step = 'step',
  SmoothStep = 'smoothstep',
  SimpleBezier = 'simplebezier',
}
```

## 注意事项

- 如果选择渲染自定义连接线组件，该值将作为 [`ConnectionLineComponentProps`](/api-reference/types/connection-line-component-props) 的一部分传递给您的组件。
