# Position

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/utils.ts/#L1)

虽然 [`PanelPosition`](/api-reference/types/panel-position) 可用于将组件放置在容器的角落，但 `Position` 枚举的定位精度较低，主要用于与边和手柄相关的场景。

```ts
export enum Position {
  Left = 'left',
  Top = 'top',
  Right = 'right',
  Bottom = 'bottom',
}
```
