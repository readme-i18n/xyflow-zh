# 移除署名

本示例展示如何从渲染器中移除 React Flow 的署名信息。

<Callout type="info">
若您考虑移除署名，我们首先想说明：

**如果您在组织中使用 React Flow 并从中获利**，我们依赖您的支持来保持 React Flow 在 MIT 许可下的开发和维护。在移除署名前，[请了解支持 React Flow 持续运行的方式](/pro)。

**您是否将 React Flow 用于个人项目？** 很好！请直接移除署名。您可以通过报告发现的错误、发送项目截图或在 [Github](https://github.com/xyflow/xyflow) 上为我们加星来支持我们。如果您未来开始通过 React Flow 盈利或在组织中使用它，我们希望您能重新添加署名或订阅我们的服务。

感谢您的支持 ✌🏻

- [xyflow 团队](https://xyflow.com/about)

</Callout>
