# 内置组件

React Flow 提供了多个内置组件，这些组件可以作为子元素传递给 [`<ReactFlow />`](/api-reference/react-flow) 组件。

## 小地图

[`MiniMap`](/api-reference/components/minimap) 提供了流程图的鸟瞰视图，使导航更加便捷，尤其适用于大型流程图。您可以通过提供 nodeColor 函数来自定义小地图中节点的外观。

<RemoteCodeViewer aspectRatio="2" route="learn/mini-map" framework="react" />

## 控制栏

React Flow 提供了一套可自定义的 [`Controls`](/api-reference/components/controls) 用于视口操作。您可以进行缩放、适配视口，并切换用户是否能移动、选择和编辑节点。

<RemoteCodeViewer aspectRatio="2" route="learn/controls" framework="react" />

## 背景

[`Background`](/api-reference/components/background) 组件为流程图添加了视觉网格图案，帮助用户保持方向感。您可以选择不同的图案变体，如需更高级的自定义，可以查看[源代码](https://github.com/xyflow/xyflow/blob/main/packages/react/src/additional-components/Background/Background.tsx)来实现自己的图案。

<RemoteCodeViewer aspectRatio="2" route="learn/background" framework="react" />

## 面板

[`Panel`](/api-reference/components/panel) 组件允许您在流程图中添加固定的覆盖层，非常适合标题、控件或任何需要保持静止的 UI 元素。

<RemoteCodeViewer aspectRatio="2" route="learn/panel" framework="react" />

## 高级用法

对于更高级的用例和自定义需求，您可以在[API 组件概览](/api-reference/components)中查看更多内置组件。
