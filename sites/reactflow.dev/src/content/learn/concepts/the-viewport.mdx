# 平移与缩放

React Flow 默认的平移和缩放行为灵感来源于
[slippy maps](https://wiki.openstreetmap.org/wiki/Slippy_map)。您可以通过拖动指针进行平移，通过滚轮进行缩放。您可以使用 `<ReactFlow />` 组件上的
[interaction](/api-reference/react-flow#interaction-props) 和
[keyboard](/api-reference/react-flow#keyboard-props) 属性轻松自定义此行为。

## 视口配置

这里我们将列出并解释其他工具使用的一些配置。

### 默认视口控制

如上所述，ReactFlow 的默认控制方式如下：

- `平移:` 指针拖动
- `缩放:` 捏合或滚轮
- `选择:` shift + 指针拖动

### 设计工具视口控制

如果您偏好 figma/sketch/设计工具的控制方式，可以将
[`panOnScroll`](/api-reference/react-flow#panonscroll) 和
[`selectionOnDrag`](/api-reference/react-flow#selectionondrag) 设为 `true`，
[`panOnDrag`](/api-reference/react-flow#panondrag) 设为 `false`：

- `平移:` 滚轮、中键/右键拖动、空格 + 指针拖动
- `缩放:` 捏合或 cmd + 滚轮
- `选择:` 指针拖动
