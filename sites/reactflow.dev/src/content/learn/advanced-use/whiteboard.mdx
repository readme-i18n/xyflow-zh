# 白板功能

React Flow 专为构建基于节点的用户界面而设计，如工作流编辑器、流程图和图表。虽然 React Flow 并非专为创建白板应用而生，但你可能希望集成常见的白板功能。这些示例展示了如何在需要为节点和边线添加注释或草图时，为你的应用添加绘图能力。

## 使用示例

### ✏️ 自由绘制 (Pro版)

在 React Flow 面板上绘制平滑曲线。适用于在现有节点周围添加注释或草图。

**功能特性:**

- 鼠标/触控绘制
- 可调节笔刷大小和颜色
- 将绘制路径转换为自定义节点

**典型用途:**

- 流程图标注
- 图表批注
- 节点周边创意草图

### 🎯 套索选择

通过绘制自由形状选区来选择多个元素，可选择是否包含部分选中的元素。

**功能特性:**

- 自由形状选区
- 元素部分选择

**典型用途:**

- 同时选择节点和标注
- 混合内容中的复杂选择

### 🧹 橡皮擦

通过"擦除"操作移除项目。使用碰撞检测机制确定删除对象。

**功能特性:**

- 基于碰撞检测的擦除功能
- 可视化橡皮擦光标

**常见用途:**

- 移除流程中的部分元素

<RemoteCodeViewer

### 📐 矩形绘制

通过点击拖拽创建矩形形状。适用于高亮区域或为节点组创建背景。

**功能特性:**

- 点击拖拽式矩形创建
- 可自定义颜色

**常见用途:**

- 创建背景容器
- 可视化分组相关节点
- 高亮图表中的特定区域

## 白板工具库

如需更完整的白板解决方案，可考虑使用专为白板应用设计的库，例如 [tldraw](https://tldraw.dev/) 或 [Excalidraw](https://docs.excalidraw.com/)。这些库提供完整的协作绘图、形状、文本等功能套件。
