# Hooks 与 Providers

React Flow 提供了多个 [hooks](/api-reference/hooks) 和一个上下文 provider，用于增强流程的功能。这些工具能帮助您更有效地管理状态、访问内部方法以及创建自定义组件。

## ReactFlowProvider

ReactFlowProvider 是一个上下文 provider，允许您从组件树的任何位置（包括 [`ReactFlow`](/api-reference/react-flow) 组件外部）访问流程的内部状态，例如节点、边和视口。通常用于应用的顶层。

以下情况可能需要使用 [`ReactFlowProvider`](/api-reference/react-flow-provider) 组件：

- 我们提供的许多 [hooks](/api-reference/hooks) 依赖此组件才能工作
- 您需要在 `ReactFlow` 组件外部访问流程内部状态
- 您在页面中处理多个流程
- 您正在使用客户端路由

## useReactFlow
