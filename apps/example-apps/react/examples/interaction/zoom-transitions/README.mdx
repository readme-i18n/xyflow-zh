---

title: 缩放过渡效果
description: 当视口缩放或移动时改变动画速度
hidden: true
---

React Flow 内置了平滑的视口过渡效果。您可以通过传递 `duration` 选项参数来为所有改变视口的函数（[`zoomIn`, `zoomOut`, `zoomTo`, `setViewport`, `fitView`, `setCenter`, `fitBounds`](/api-reference/types/react-flow-instance#viewport)）定义动画持续时间。
