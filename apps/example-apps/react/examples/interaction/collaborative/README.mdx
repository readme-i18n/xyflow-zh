本示例展示如何通过 React Flow 和 [yjs](https://github.com/yjs/yjs) 构建多用户协作图表。当前页面中所有用户都能实时看到图表变更。要查看实际效果，您可以在[新标签页](/examples/interaction/collaborative)或不同设备/浏览器中打开此页面。
该示例可作为构建多用户流程图和编辑器的开发起点。
