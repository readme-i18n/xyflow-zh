---

title: 验证
description: 检查新连接是否有效并应被添加
---

自定义节点需要至少包含一个 [`Handle` 组件](/api-reference/components/handle)才能建立连接。您可以通过向 ReactFlow 组件传递验证函数 [`isValidConnection`](/api-reference/react-flow#isvalidconnection-1) 来检查新连接是否有效并应被添加。
