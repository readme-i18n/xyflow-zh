---

title: 拖放功能
description: 使用原生 HTML 拖放 API 或 react-draggable 在 React Flow 面板外实现拖放操作。
---

拖放式用户界面在基于节点的工作流编辑器中非常常见。React Flow 面板外的拖放行为并非内置功能，但可以通过原生 [HTML Drag and Drop API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_Drag_and_Drop_API)（本示例采用此方案）或第三方库如 [react-draggable](https://github.com/react-grid-layout/react-draggable) 来实现。
