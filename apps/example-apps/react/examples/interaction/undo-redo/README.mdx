本示例演示如何为 React Flow 图表实现撤销与重做功能。用户可以追踪并回退移动、添加或删除节点与连线时的变更。该实现采用基于快照的方法，使用 `useUndoRedo` 钩子来管理历史状态与未来状态，允许用户通过按钮点击或键盘快捷键（Ctrl+Z 和 Ctrl+Shift+Z）在编辑历史记录中导航。
