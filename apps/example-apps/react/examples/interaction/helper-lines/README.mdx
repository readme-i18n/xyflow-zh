本示例演示如何实现对齐辅助线功能。当您拖动节点时，这些视觉引导线会自动出现以帮助节点间对齐，同时吸附机制确保精确定位。

该实现同时处理水平和垂直对齐，并能正确响应视口的变换和缩放，确保即使在放大或平移时引导线仍保持精确。
