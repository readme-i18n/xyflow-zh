本示例演示如何自动排列节点。它引入了一个可复用的 `useAutoLayout` 钩子，并展示了三种可在运行时切换的流行布局引擎：[dagre](https://github.com/dagrejs/dagre)、[d3-hierarchy](https://github.com/d3/d3-hierarchy) 和 [elk](https://github.com/kieler/elkjs)。

### 关于此专业版示例

- 依赖项: [@xyflow/react](https://www.npmjs.com/package/@xyflow/react), [@dagrejs/dagre](https://www.npmjs.com/package/@dagrejs/dagre), [d3-hierarchy](https://www.npmjs.com/package/d3-hierarchy), [elkjs](https://www.npmjs.com/package/elkjs)
- 包含可配置的 `useAutoLayout` 钩子，可在您的应用中复用
- 提供不同布局算法的对比及交互式控制
