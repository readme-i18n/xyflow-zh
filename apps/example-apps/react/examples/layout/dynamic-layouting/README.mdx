本示例创建了一个自组织图形系统，节点会自动排列成树状结构。主要特性包括：

- **节点创建**：用户无需手动放置或连接节点，只需点击占位元素即可添加新节点
- **多重交互点**：通过点击节点添加子节点、将占位符转换为节点，或使用边上的"+"按钮在现有节点间插入新节点
- **智能布局算法**：当图形变化时，基于`d3-hierarchy`的自动布局钩子会重新计算所有位置
- **平滑过渡**：节点会动画过渡到新位置，即使复杂度增加也能保持视觉清晰度
