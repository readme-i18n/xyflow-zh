本示例展示如何将 [dagre.js](https://github.com/dagrejs/dagre) 与 React Flow 集成来创建简单树状布局。如果您需要更高级的布局库，[d3-hierarchy](https://github.com/d3/d3-hierarchy) 或 [elkjs](https://github.com/kieler/elkjs) 是 dagre 的良好替代方案。
