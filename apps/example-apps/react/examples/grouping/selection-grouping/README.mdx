本示例演示如何在 React Flow 中创建动态节点分组。您可以通过按住 Shift 键并点击来选择多个节点，然后使用出现的工具栏中的"分组节点"按钮将它们组合在一起。当选中分组节点时，会出现"取消分组"按钮，用于分离所有子节点。分组节点可完全调整大小，并会根据其子节点自动调整尺寸。
