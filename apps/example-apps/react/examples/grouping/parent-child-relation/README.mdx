本示例演示如何动态创建节点间的父子关系。您可以将节点拖拽到分组节点上使其成为子节点，或使用工具栏的"detach"按钮移除该关系。该实现处理了将绝对位置转换为父节点相对坐标所需的复杂位置计算。
