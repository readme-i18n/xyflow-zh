本示例演示如何在不同图形布局之间创建流畅的动画过渡效果。当点击"切换布局"按钮时，节点会平滑地动画过渡到新位置，提供了一种视觉上吸引人的布局切换方式。

### 关于此专业版示例

- 依赖项: [@xyflow/react](https://www.npmjs.com/package/@xyflow/react), [d3-timer](https://www.npmjs.com/package/d3-timer)
- 实现可复用的 `useAnimatedNodes` 钩子函数实现平滑过渡
- 使用线性插值算法实现节点位置动画
- 展示水平与垂直布局之间的位置补间动画
- 提供简单的切换控件来改变不同布局
- React Flow 自版本 12 起支持服务端渲染
