本示例展示如何渲染流程图常用的圆形、菱形或六边形等不同形状的[自定义节点](/learn/customization/custom-nodes)。同时演示了如何创建侧边栏组件、自定义缩略图节点以及用于更改形状颜色的节点工具栏。

该示例使用中心化的 Shape 组件，根据形状类型渲染不同的 SVG 路径。通过定义节点数据 `{ type: 'shape', data: { type: 'diamond', color: '#ff0071' }}`，您可以控制渲染哪种形状及其外观，同时仅使用单一节点类型实现。
