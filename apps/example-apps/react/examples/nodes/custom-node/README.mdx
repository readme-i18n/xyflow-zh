title: 自定义节点
description: 在节点内显示任意内容
---

创建自定义节点就像构建一个常规的 React 组件并将其传递给 `nodeTypes` 一样简单。由于它们是标准的 React 组件，您可以显示任何内容并实现所需的任何功能。此外，您还可以访问一系列属性，这些属性允许您扩展和自定义默认节点行为。更多详细信息，请参阅我们的[自定义节点指南](/learn/customization/custom-nodes)。
