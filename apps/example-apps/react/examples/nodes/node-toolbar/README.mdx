---

title: 节点工具栏
description: 在选中节点旁显示带按钮的工具栏
---

对于许多类型的应用程序而言，当选中节点时显示工具栏或一组控件会非常实用。事实上这个功能如此实用，我们专门构建了 [`<NodeToolbar />`](/api-reference/components/node-toolbar) 组件，让您能轻松地为自定义节点添加此功能！

<Callout type="info">
  工具栏内容不会随着流程图的缩放而改变大小：这意味着它应该始终保持可见。
</Callout>
