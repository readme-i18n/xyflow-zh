---

title: 自定义边线
description: 创建具有特殊路径或沿边控件的边线
---

React Flow 提供四种不同的边线类型 - `default` (贝塞尔曲线)、`straight` (直线)、`step` (阶梯线) 和 `smoothstep` (平滑阶梯线)。若需要特殊边线路径或沿边控件，您也可以创建[自定义边线](/api-reference/types/edge-props)。本示例将演示如何实现带按钮的边线、双向边线以及自连接边线。所有示例均使用[BaseEdge](/api-reference/components/base-edge)组件作为辅助工具。
