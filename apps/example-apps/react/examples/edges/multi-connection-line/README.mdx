---

title: 多重连接线
description: 一次性从任意选中的节点绘制多条连接线。
---

React Flow 通常一次只允许创建一条连接。本示例基于[自定义连接线](/examples/edges/custom-connectionline)示例，展示如何一次性从任意选中的节点绘制多条连接线。

<Callout type="info">
  请注意 `onConnect` 处理函数。如果忘记包含它，即使选中了多个节点也只会创建一条连接！
</Callout>

</Callout>
