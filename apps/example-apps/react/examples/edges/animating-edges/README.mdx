React Flow 为默认边缘类型提供了简单的内置动画，但通过使用自定义边缘可以实现更高级的动画效果。以下是展示边缘路径不同应用方式的示例集合。

## SVG元素动画

可以使用 `<animateMotion />` 元素实现SVG元素沿路径运动。此示例创建了一个自定义边缘，使圆形沿边缘路径产生动画效果。

## 其他节点动画

若需实现比SVG更复杂的边缘动画，可以使用Web Animations API。本示例演示如何通过`offsetPath`属性和`offsetDistance`属性动画，使流程中的另一个节点沿边缘路径运动。

<RemoteCodeViewer route="examples/edges/animating-edges" framework="react" />

此示例中有几个重要细节需要注意：

- 动画节点在运行动画时将其[`draggable`](/api-reference/types/node#draggable)属性设为`false`，防止用户移动节点导致动画路径中断

- 动画路径和动画本身分别在独立的`useEffect`钩子中设置。这样即使边缘路径重新计算（例如拖动源节点或目标节点时），动画也能保持流畅播放
