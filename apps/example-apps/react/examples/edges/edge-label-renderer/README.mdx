---

title: Edge Label Renderer
description: 在边缘上方以 div 形式渲染边缘标签
---

若需在边缘元素中脱离 SVG 环境，可使用 [`<EdgeLabelRenderer />`](/api-reference/components/edge-label-renderer) 组件。这是一个门户组件，允许您以 div 形式在边缘上方渲染标签。如需添加鼠标交互，需在标签上设置 `pointer-events: all` 样式。
