---

title: 重新连接边线
description: 通过点击拖动将现有边线从一个连接点移动到另一个
---

当使用 [`onReconnect` 处理函数属性](/api-reference/react-flow#edges)时，可通过拖动边线到其他连接点实现重新连接。该处理函数会在边线被放置到新连接点后触发。您可以使用 [`reconnectEdge`](/api-reference/utils/reconnect-edge) 辅助函数来相应更新边线状态。

以下属性共同作用决定边线是否可更新：

- 默认情况下 [`edgesReconnectable`](/api-reference/react-flow#edgesReconnectable) 设置为 `true`

- 但要使边线实际可拖动，还必须定义 [`onReconnect`](/api-reference/react-flow#onReconnect) 处理函数

- 单个边线可通过设置其 [`updatable`](/api-reference/types/edge#updatable) 属性来细化或覆盖 [`edgesReconnectable`](/api-reference/react-flow#edgesReconnectable) 属性

  - `true` 表示即使 [`edgesReconnectable`](/api-reference/react-flow#edgesReconnectable) 为 false 边线仍可更新

  - `"source"` 或 `"target"` 表示边线仅能从源或目标连接点更新，不受 [`edgesReconnectable`](/api-reference/react-flow#edgesReconnectable) 属性影响

  - `false` 表示即使 [`edgesReconnectable`](/api-reference/react-flow#edgesReconnectable) 设为 `true` 边线也不可更新
