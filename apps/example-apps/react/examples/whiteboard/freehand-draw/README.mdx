本示例展示了如何在 React Flow 应用中实现手绘功能。用户可以在画布上绘制平滑曲线，该功能适用于添加注释或在现有节点周围进行草图绘制。绘制内容会被转换为自定义节点，这些节点可以像常规节点一样进行样式设置和操作。
