本示例展示如何创建一个橡皮擦工具，通过擦除操作来删除节点和连线。
该工具由两部分组成：

1. 处理擦除逻辑和橡皮擦轨迹渲染的 `Eraser` 组件
2. 响应 `toBeDeleted` 标志的自定义 `ErasableNode` 和 `ErasableEdge` 组件

判断轨迹是否与节点相交相对简单，但检测轨迹与连线的交点则更为复杂：
我们通过 SVG 路径元素的 [`getPointAtLength`](https://developer.mozilla.org/en-US/docs/Web/API/SVGGeometryElement/getPointAtLength) 方法沿连线采样点，
构建一个可用于检测与橡皮擦轨迹交点的折线。
这是在性能与精度之间的权衡 - 您可以调整 `sampleDistance` 变量来观察它对橡皮擦轨迹的影响。
