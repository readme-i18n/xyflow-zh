本示例演示了如何在面板上使用套索选择工具选取多个项目。点击并拖动创建选择区域，该区域内的所有项目将被选中。

- **套索绘制**：使用指针事件自由绘制选择区域
- **部分/完全选择**：切换选择被套索部分或完全包围的节点
- **视觉反馈**：绘制选择区域时提供实时视觉反馈
- **画布渲染**：使用HTML5 Canvas实现流畅的绘制性能
