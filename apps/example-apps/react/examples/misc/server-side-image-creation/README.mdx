本示例演示了如何在服务器端创建 React Flow 图形的静态图像。应用程序允许您在屏幕左侧与节点交互，同时自动在右侧生成并显示对应的流程图图像。该实现结合了 React Flow 新增的静态渲染能力与 Puppeteer 进行截图捕获。
