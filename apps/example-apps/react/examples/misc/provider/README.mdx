---

title: ReactFlowProvider
description: 用于在页面中处理多个流程图，确保每个流程图拥有独立的存储实例
hidden: true
---

若需在单个页面中处理多个流程图，或使用客户端路由时，必须用 [`ReactFlowProvider`](/api-reference/react-flow-provider) 包裹每个流程图组件，以确保每个流程图拥有独立的存储实例。
当需要在 `ReactFlow` 组件外部访问内部状态时，也必须使用 `ReactFlowProvider`。
