本示例展示如何使用 [html-to-image](https://github.com/bubkoo/html-to-image) 将流程图下载为图片。

<Callout type="info">
  本示例中使用的 [html-to-image](https://www.npmjs.com/package/html-to-image) 包版本已锁定为 `1.11.11`，这是该包最新可用的工作版本。`1.11.11` 之后的近期版本无法正确导出图片，Github 上已有相关 [issue](https://github.com/bubkoo/html-to-image/issues/516) 记录。
</Callout>
